Database Schema

conversations

id INTEGER PRIMARY KEY
participant1_id INTEGER NOT NULL
participant2_id INTEGER NOT NULL
created_at TEXT DEFAULT datetime('now')
UNIQUE(participant1_id, participant2_id)

messages

id INTEGER PRIMARY KEY
conversation_id INTEGER NOT NULL
sender_id INTEGER NOT NULL
content TEXT (nullable)
image TEXT (nullable)
is_read INTEGER DEFAULT 0
created_at TEXT DEFAULT datetime('now')

Known Bugs

⚠ 401 on POST via Cloudflare Tunnel
POST requests to API return 401 when accessed via Cloudflare tunnel URL. The session cookie is tied to localhost domain. Fix: ensure cookie domain handling works with tunnel, or login through tunnel URL.
⚠ NOT NULL constraint on messages.content
Sending a message with only text fails with SQLITE_CONSTRAINT_NOTNULL. The INSERT tries to insert empty string instead of NULL for content.
⚠ Image upload missing Postgres migration
The image column migration for messages table only handles SQLite (PRAGMA). Postgres path is missing.
